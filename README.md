Live Demo: https://mad-et-clone.herokuapp.com/
## Getting Started

First, install the dependencies by running:

```bash
npm install
# or
yarn 
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
